Cheese Website

For my university project, I was tasked to develop an e-commerce website focused on cheese, built using PHP, JavaScript, HTML, CSS, and SQL. The website includes a fully functional admin section where administrators can manage cheese products, allowing them to add new items and update names, prices, and other details. All data is securely connected to an SQL database for efficient management and storage. Users can add items to their basket, with the total automatically calculated based on the quantities and prices. When proceeding to checkout, the order and the assosiated user who placed the order is logged in the SQL database.

The site also features a search functionality enhanced by AJAX, offering real-time auto-completion for quicker and more intuitive navigation. I took special care to incorporate accessibility features, addressing factors such as vision impairment, hearing impairment, mobility issues, cognitive impairments, dyslexia, and neurodivergence, ensuring the platform is inclusive for all users. Additionally, I implemented key security measures, including protection against cross-site scripting (XSS) and SQL injection, to safeguard user data and maintain the integrity of the site. This project highlights a balance of functionality, user experience, and security in web development.
